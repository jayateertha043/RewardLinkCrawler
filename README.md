## RewardLinkCrawler

> Takes waybackurl input of https://www.rewardlink.io/ url & outputs available balances for gift urls obtained in stdout & csv file

### Installation
```
go build .
```

### Usage
```
waybackurls https://www.rewardlink.io/|RewardLinkCrawler
```
